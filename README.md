# LegalX AI Knowledge Centre

## Overview
LegalX AI is an AI-powered legal knowledge platform that helps users understand important Indian legal acts using Retrieval-Augmented Generation (RAG).

## Features
- Legal document summarization
- Key information extraction
- AI Legal Assistant
- Multi-document search
- Audio summaries
- Multi-language support

## Technologies Used
- Streamlit
- Python
- LangChain
- Google Gemini
- ChromaDB
- HuggingFace Embeddings
- gTTS

## Legal Acts Covered
- POCSO Act
- RTI Act
- Consumer Protection Act
- Cybercrime Laws

## Run Locally

pip install -r requirements.txt

streamlit run app.py
